# vidsrc [shell script]

A minimal movie/tv series streaming bash script 

## How to install ?

```sh
git clone https://github.com/isg32/vidsrc-shell/
```
## Make it executable first !
```sh
cd vidsrc-shell
chmod +x vidsrc.sh
```

## Usage

```sh
./vidsrc.sh '[movie/TV.S name]'
```

## Note:- enter the name in single inverted commas
